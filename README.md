# TwitterPersonality
Predicts the Twitter user personality using his tweets
Personality of auser is predicted using OCEAN properties.
Refer these links for OCEAN properties 
https://en.wikipedia.org/wiki/Big_Five_personality_traits 
http://www.outofservice.com/bigfive/

File Tweet_extraction extracts tweets from each individual user and gets 200 tweets per user. 
Celebrities.txt file contains all the twitter usernames for which you want to get tweets.
File get_personality.py gives the OCEAN values for each user.
